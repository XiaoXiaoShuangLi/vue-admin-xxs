# 引言  
Vue Admin 是一款基于Ant Vue 框架之上, 模仿腾讯云UI界面写的后台.

PS: 如果在使用过程失效可以到[ISSUES](https://github.com/XiaoXiaoShuangLi/vue-admin-xxs/issues) 提问题我们会即时回复。


# 运行环境
- `Vue`
- `WebStorm`
- `TypeScript`
- 推荐 `Mac OS`或 `linux` 、 `Windows`

# 后缀规范
module 表示包名后缀
component 表示组件后缀
